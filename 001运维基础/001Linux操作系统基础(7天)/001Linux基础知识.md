## 1️⃣ Linux简介

🖥️ **什么是 Linux**
 Linux 是一个类 Unix 的自由与开源操作系统，由 **Linus Torvalds** 于 1991 年创建。
 它以 **稳定、安全、高效** 而著称，广泛用于服务器、嵌入式系统、超级计算机等领域。

🔹 **主要特点：**

- 🧱 开源自由，可自由修改与分发
- ⚙️ 稳定高效，适合长期运行
- 🔐 安全性高，权限与用户机制完善
- 💻 跨平台支持，从服务器到嵌入式设备
- 🌐 社区活跃，软件生态丰富

🔹 **常见发行版：**

| 类型       | 代表系统            | 说明               |
| ---------- | ------------------- | ------------------ |
| 🌿 Debian系 | Ubuntu、Kali Linux  | 用户友好，软件丰富 |
| 🔥 RedHat系 | CentOS、Rocky Linux | 企业级服务器常用   |
| 🌀 独立系   | Arch Linux、Gentoo  | 灵活、适合高级用户 |

🧠 **内核与操作系统的关系：**

- **内核（Kernel）**：系统核心，负责进程调度、内存与硬件管理。
- **操作系统（OS）**：包含内核 + 各种系统工具与库。

📘 **查看内核版本：**

```bash
uname -r
# 示例输出：5.15.0-91-generic
```

## 2️⃣ Linux命令行基础

💡 **命令行与 Shell**
 Shell 是与操作系统交互的命令解释器，常见类型有：

- **Bash**（默认，最常见）
- **Zsh**（美观，自动补全强）
- **Fish**（智能提示，用户体验优秀）

📘 **查看当前 Shell：**

```bash
echo $SHELL
```

------

### 🧭 常用基础命令

| 命令  | 功能说明         | 示例                            |
| ----- | ---------------- | ------------------------------- |
| `ls`  | 列出目录内容     | `ls -al` 查看所有文件（含隐藏） |
| `cd`  | 切换目录         | `cd /etc`                       |
| `pwd` | 显示当前目录路径 | `pwd`                           |
| `cp`  | 复制文件/目录    | `cp file.txt /tmp/`             |
| `mv`  | 移动或重命名     | `mv old.txt new.txt`            |
| `rm`  | 删除文件/目录    | `rm -r dir/`                    |
| `man` | 查看命令手册     | `man ls`                        |

💡 **小技巧：**

- 使用 `Tab` 自动补全命令或路径。
- `Ctrl + C` 终止当前命令。
- `Ctrl + R` 搜索历史命令。

## 3️⃣ Linux文件系统

📂 **文件系统结构（树形结构）**

```bash
/
├── bin/      # 系统命令
├── etc/      # 配置文件
├── home/     # 用户主目录
├── var/      # 可变数据（日志、缓存）
├── usr/      # 用户安装的软件
├── tmp/      # 临时文件
└── root/     # 超级用户的家目录
```

🧱 **文件路径类型：**

- **绝对路径**：以 `/` 开头，例如 `/home/user/file.txt`
- **相对路径**：相对于当前目录，如 `../` 表示上级目录

📘 **查看磁盘空间与目录结构：**

```bash
df -h       # 查看磁盘使用情况
du -sh *    # 查看当前目录下各文件大小
tree /etc   # 树形展示目录（需安装 tree）
```
